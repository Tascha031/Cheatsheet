Table of HTML elements.

Tags Name Description
<h1> - <h6> Heading Element h1 make headings big, h6 make headings small
<p> Paragraph Element Contains and displays a text
<ul> Unordered List Element Puts dots in front of list items
<em> Emphasis Element Make text look italics
<strong> Strong Element Make text look bold
<div> Div Element Division: divides a HTML document in sections
<span> Span Element Container to seperate text, from larger text
<br> Line Break Element Element Create a line break in text
